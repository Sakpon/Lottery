# แพลตฟอร์มวิเคราะห์สลากกินแบ่งรัฐบาลไทย

แพลตฟอร์มวิเคราะห์และทำนายหวยไทย (สลากกินแบ่งรัฐบาล) โดยอาศัยสถิติจากผลรางวัลย้อนหลัง
รันบน **Cloudflare Workers + Pages + D1** และเก็บโค้ดบน **GitHub**

> ⚠️ **ข้อความปฏิเสธความรับผิดชอบ**: ผลการทำนายเป็นเพียงการวิเคราะห์ทางสถิติของข้อมูลย้อนหลังเท่านั้น
> การออกรางวัลสลากกินแบ่งรัฐบาลเป็นการสุ่ม ไม่สามารถคาดเดาได้จริง โปรดเล่นอย่างมีสติ

## โครงสร้างโปรเจกต์

```
lottery/
├── web/                    # Cloudflare Pages (หน้าเว็บภาษาไทย Mobile-first)
├── workers/
│   ├── scraper/            # Scheduled Worker ดึงข้อมูลจาก sanook.com/lotto
│   ├── api/                # HTTP API serving draws/stats/predictions
│   └── predictor/          # โมเดลทำนาย (frequency, gap, Markov, ensemble)
├── data/
│   ├── schema/             # D1 schema SQL
│   └── seed/               # ข้อมูลเริ่มต้นย้อนหลัง
├── scripts/                # สคริปต์ backfill / maintenance
└── .github/workflows/      # CI/CD deploy workers + pages
```

## โมเดลการทำนาย

1. **Frequency (ความถี่)** — เลขที่ออกบ่อยในช่วง N งวดล่าสุด (ร้อน)
2. **Gap Analysis (ช่วงห่าง)** — เลขที่ไม่ออกมานาน (เย็น/ค้างนาน)
3. **Digit-position** — ความถี่ของตัวเลขในแต่ละตำแหน่ง
4. **Markov Chain** — ความน่าจะเป็นของเลข 2 ตัวท้าย ตามเลขงวดก่อน
5. **Ensemble** — ถ่วงน้ำหนักผลรวมจากทุกโมเดล พร้อม confidence score
